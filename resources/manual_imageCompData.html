face


Aim the turtle to face a x-y coordinate. Enter numbers for the coordinates.

go forward


Move the turle in the direction that it is pointing.
You must set the step size with or the turtle will not move.

go invisible


This commmands makes the turtle disappear from the world. Though you cannot see the turtle, it is still executing the commands.

go visible


This commands makes the turtle visible in the world.

left [number]


Turns the turtle number degrees to the left of the current heading.

place measure point


Puts a mark on the world of where the turtle currently is located.
The measure points can be read by the graphing tools in ViMaP. This command can be undone by .

pen down


Leaves a pen-trail wherever the turtle travels in the world.

pen up


The turtle will no longer leave a pen-trail. But previous pen marks will persist in the world.

right [number]


Turns the turtle number degrees to the right of the current heading. It is the opposite command to

set heading [number]


Enter a number in the textbox. The turtle will read that number as degrees and turn and face in that direction.
In ViMaP heading 0 is north, heading 90 is east, heading 180 is south, and heading 270 is west.

set random heading


Set the turtle's heading, to a random number.
The new heading will be between 0 and the number you choose.

set shape


Set the shape of the turtle, to the shape in the dropdown menu.

stamp


Stamp an outline of the turtle onto the background.

step size minus


Set step size for the turtle, equal to the current step size minus a number you enter.

step size plus


Set step size for the turtle, equal to the current step size plus a number you enter.
